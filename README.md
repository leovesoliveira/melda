# Melda

Um jogo de qualidade duvidosa baseado em um jogo de nome parecido.

## Comandos

| Ação                  | Tecla                            |
|-----------------------|----------------------------------|
| Mover para Cima       | `W` ou `Seta para Cima`          |
| Mover para Baixo      | `S` ou `Seta para Baixo`         |
| Mover para Esquerda   | `A` ou `Seta para Esquerda`      |
| Mover para Direita    | `D` ou `Seta para Direita`       |
| Atirar                | `Espaço` (Requer arma e munição) |
| Pausar / Menu         | `Esc`                            |
| Selecionar no Menu    | `Enter`                          |
| Reiniciar (Game Over) | `Enter`                          |
| Salvar Jogo           | `F1`                             |
